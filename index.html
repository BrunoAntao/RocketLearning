Importância do JavaScript


No paradigma das tecnologias web é notório que o JavaScript é usado em 94,5% dos websites e como uma linguagem que executa no browser, o JavaScriptajuda os programadores a tornar as páginas interativas e dinâmicas, mais ainda, pode combinar-se JavaScript com CSS e HTML para tornar as páginas apelativas mantento a sua aparência em qualquer browser e em qualquer dispositivo.



Por outro lado, o JavaScript pode ser utilizado, em conjunto com Node JS, como uma linguagem de servidor. O Node JS, por sua vez, executa o código através do motor V8 do Chrome, tornando-se uma ferramenta poderosa e indispensável nos dias de hoje, para além de permitir que os web developers possam escrever código para servidor também em JavaScript

O que é o JavaScript?


JavaScript é uma linguagem de programação usada maioritariamente nos browsers web e que permite implementar interações complexas, por exemplo: atualizar o conteúdo da página dinâmicamente, animar a informação mostrada na página, animar gráficos 2D/3D, entre outras.

Os primeiros traços de JavaScript foram desenvolvidos no final da década de 90, altura em que as páginas web eram estáticas, com pouca interação para além de clicar e carregar novas páginas. Nessa altura, e pela primeira vez, o JavaScript permitiu animar e adaptar o conteúdo da página.



Durante muitos anos o JavaScript só funcionou num número limitado de browsers, forçando os programadores que queriam desenvolver sites a escolher um conunto de browsers em vez de outros o tornou a Internet menos universal e pouco acessível. Só em 1999 o JavaScript foi adotado globalmente. Ainda assim, a compatibilidade dos browsers permaneceu um problema por mais de uma década.
